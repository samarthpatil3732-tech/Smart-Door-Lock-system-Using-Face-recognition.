Project Overview

This Smart Door Lock Security System uses Face Recognition + OTP verification to enhance home and office security. It combines computer vision, email alerting, and Arduino-controlled locking. The camera scans the face for 10 seconds and verifies users using a trained dataset. If the face does not match, the system sends an intruder image to the ownerâ€™s email. If the face matches, OTP authentication is required for final unlocking, making it more secure than traditional locks.
